Here are a simple examples how work API.

Get all articles

URL: host/article

link

Article adding

URL: host/temp/articles (POST)

Data will be saved into temp.

Request example: 

	$.ajax({
		url: 'http://localhost:3000/temp/articles',
		type: 'POST',
		dataType : "json",
		data: {
			media: {
				content: 'Some html' // Not empty
			},
			incident: {
				time: 10928374, // 0 <= time <= current time
				type: '', // empty foe this moment
				coordinates: {
					lat: 0, // 0 <= lat
					lon: 0 // 0 <= lon
				},
				title: ''
			}
		}
	});
